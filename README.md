# The Open Source Operating System
The Persian OS operating system is made in Iran. This operating system is a free and open source operating system, written in Python, and the source is provided to users. This operating system can be an alternative to Windows Mac and Linux.
# How To Clone
You can clone in the following command
```bash
git clone https://github.com/mahdianimator/persiaos
```
```bash
cd persiaos
```
# Fix Error
You can fix the error with the following command
```bash
mkdir user
```
Before that, you need to install prerequisites
```bash
pip install tk
```
```bash
pip install pyqt5
```
